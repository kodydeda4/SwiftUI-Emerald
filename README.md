## About Emerald

Emerald is a Tiling Window Manager for macOS built with the [SwiftUI](https://developer.apple.com/xcode/swiftui/) framework.  It attempts to provide a user-friendly experience for customizing [Yabai](https://github.com/koekeishiya/yabai/wiki) & [SKHD](https://github.com/koekeishiya/skhd). The project is currently inactive.
